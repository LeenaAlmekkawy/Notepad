# Notepad Application

This project is a lightweight, multi-tabbed Notepad application built using Java and JavaFX. It provides essential text editing features, a simple interface, and customization options for an enhanced user experience.

---

## Features

### File Management
- **New**: Create a new text document.
- **Open**: Load existing text files from your local machine.
- **Save**: Save your text to a file in `.txt` format.

### Editing Tools
- **Undo/Redo**: Quickly revert or reapply changes.
- **Cut/Copy/Paste**: Standard clipboard operations for text editing.
- **Delete**: Remove selected text.
- **Select All**: Quickly highlight all text.

### Font and Style
- Choose from a variety of fonts such as `Times New Roman`, `Arial`, `Comic Sans MS`, and more.
- Adjust font sizes from 5 to 100 for better readability or presentation.

### Multi-tab Interface
- Create multiple tabs to work on different text files simultaneously.
- Tabs can be added dynamically and managed independently.

### Status Bar
- Displays real-time updates for:
  - Current line and column numbers.
  - Total character count (excluding spaces).

### Help
- **About**: Provides information about the app and its features.
